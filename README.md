# ⚡ Skills Icons Generator

A clean, interactive tool to build your tech stack visually. Search for icons and instantly generate colorful, brand-accurate Markdown for your GitHub profile README.

## 🚀 How to Use

1.  **Search**: Use the interactive search bar to find your tools. It supports smart aliases (e.g., searching "vscode" finds "Visual Studio Code" and "photoshop" finds "Adobe Photoshop").
2.  **Preview**: Watch your stack grow in real-time. Icons are rendered using their official brand colors via Iconify.
3.  **Copy**: Click the copy button to grab the generated Markdown code and paste it directly into your GitHub `README.md`.

---

## ✨ Features

- **Interactive Search**: Powered by Fuse.js for smart, fuzzy-matching results.
- **Iconify Engine**: High-quality SVG icons that provide brand-accurate colors without bloating your project.
- **Dynamic URL Sync**: Your selected stack is automatically reflected in the URL, making it easy to share your configuration or save it for later.
- **Zero Configuration**: No database required; the app runs entirely on the client side.

## 🛠️ Installation & Setup

```bash
# 1. Clone the repository
git clone [https://github.com/your-username/skills-icons-generator.git](https://github.com/your-username/skills-icons-generator.git)

# 2. Enter the directory
cd skills-icons-generator

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```
